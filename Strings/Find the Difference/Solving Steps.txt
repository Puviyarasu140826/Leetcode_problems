Find the Difference is the 389th problem on leetcode.

Problem Statement:
      You are given two strings s and t.String t is generated by random shuffling string s and then add one more letter at a random position.Return the letter that was added to t.

Steps to solve:
  1. Iterate the string and increase the count variable for every character in string.
  
  2. Then decrease the count for another string.
  
  3. Return the count variable for finding the difference
